Complexity Analysis

Time Complexity: O(N)
Every element from the array is added to the Deque exactly once and removed at most once. Amortized time per element is O(1).

Space Complexity: O(K)
The Deque `q` stores at most `k` elements at any given time (the indices of elements in the current window). The result array `r` takes O(N - K + 1) space.
