Complexity Analysis

Time Complexity: O(N)
The algorithm traverses the linked list at most once up to the right node. Reversing the sublist is done in a single pass of right - left iterations.

Space Complexity: O(1)
The reversal is done in-place. We only use a few extra pointers (dummy, prev, curr, nxt) regardless of the size of the linked list.
