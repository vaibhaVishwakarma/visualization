Complexity Analysis

Time Complexity: O(N²)
Where N is the number of rows in the triangle. The algorithm visits every element in the triangle exactly once, starting from the second-to-last row up to the top. There are N*(N+1)/2 elements, making the time complexity proportional to N².

Space Complexity: O(N²)
The provided solution explicitly builds a full 2D `dp` array of the same shape as the input triangle, requiring O(N²) space. (Note: This can be optimized to O(N) by maintaining only a single 1D array representing the row directly below).
