Complexity Analysis

Time Complexity: O(M × N)
The algorithm initializes the first row and column, then uses a nested loop to visit every other cell exactly once. Each cell computation is an O(1) addition.

Space Complexity: O(M × N)
We construct a 2D vector `dp` of size M rows and N columns to store the number of paths to every cell. (Note: Space can be optimized to O(N) by only storing the previous row).
