Complexity Analysis

Core Logic:
Manacher's Algorithm transforms the string by inserting a special character (e.g., `#`) between every character to handle both even and odd length palindromes uniformly. It utilizes a `P` array to store the palindrome radius at each center. By maintaining the rightmost palindrome boundary `R` and its center `C`, it uses previously computed values in `P` (via the `mirror` index) to skip redundant character comparisons.

Time Complexity: O(N)
Although there is a nested `while` loop, the right boundary `R` strictly increases. Every successful character comparison inside the `while` loop expands `R`. Since `R` can at most reach `2N+3`, the inner loop runs at most `O(N)` times overall. Thus, the entire algorithm runs in linear time.

Space Complexity: O(N)
We create a transformed string `T` of size `2N + 3` and a `P` array of the same size.
