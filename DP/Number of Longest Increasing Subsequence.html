Combination Tracking Rules

• If length increases (`dp[i] < dp[prev] + 1`):
  A strictly longer path has been opened up. Overwrite length and reset combinations from the contributor branch: `cnts[i] = cnts[prev]`.
• If length ties (`dp[i] == dp[prev] + 1`):
  An alternate routing can match the current max length record. Accumulate the total counts: `cnts[i] += cnts[prev]`.
