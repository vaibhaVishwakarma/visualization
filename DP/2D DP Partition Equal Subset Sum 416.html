Complexity Analysis

Time Complexity: O(N × Target)
Where N is the number of elements and Target is half of the total sum. The algorithm iterates through every element for every possible sum up to the target, performing O(1) checks per cell.

Space Complexity: O(N × Target)
We construct a 2D boolean `dp` array of size `N x (Target + 1)` to store whether a specific sum is achievable using a subset of the elements up to index `i`. (Note: Can be optimized to O(Target) by only keeping the previous row).
