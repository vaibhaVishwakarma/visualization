Complexity Analysis

Time Complexity: O(N)
A single loop runs from `i = 2` to `n-1`. In each iteration, we perform an O(1) maximization `max(dp[i-1], nums[i] + dp[i-2])`.

Space Complexity: O(N)
We use an array (vector) `dp` of size `n` to store the maximum loot up to each house. (Note: This can be optimized to O(1) space by only storing the previous two states).
