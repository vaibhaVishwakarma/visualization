Complexity Analysis

Time Complexity: O(N)
The algorithm runs the linear house robber routine twice. Each routine iterates through approximately N-1 elements. Total time is linear.

Space Complexity: O(1)
By using `prev1` and `prev2` to keep track of the max loot from the last two houses, the space is reduced to constant O(1) memory, instead of an O(N) DP array.
