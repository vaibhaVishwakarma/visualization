Complexity Analysis

Time Complexity: O(N)
A single loop runs from `k = 3` to `n`. In each iteration, we perform an O(1) addition `dp[k-1] + dp[k-2]` and an O(1) push_back.

Space Complexity: O(N)
We use an array (vector) `dp` of size `n + 1` to store the number of ways to reach each step. (Can be optimized to O(1) by only keeping the last two variables, but the code explicitly builds the array).
