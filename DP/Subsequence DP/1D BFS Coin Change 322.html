Complexity Analysis

Time Complexity: O(S * N)
Where S is the amount and N is the number of coin denominations. In the worst case, the BFS explores every amount from 0 to S, and at each amount, it loops through all N coins.

Space Complexity: O(S)
The `visited` array and the Queue take up space proportional to the target amount `S`.
