Complexity Analysis

Core Logic:
The smallest possible answer is the maximum element in the array (if k == nums.length). The largest possible answer is the sum of all elements (if k == 1). We binary search the answer in the range [max(nums), sum(nums)].

Time Complexity: O(N log S)
Where N is the number of elements and S is the sum of all elements in the array. The binary search takes O(log S) steps. Verifying if we can split the array takes O(N) time.

Space Complexity: O(1)
We only use primitive variables, no extra arrays are allocated for the computation.
