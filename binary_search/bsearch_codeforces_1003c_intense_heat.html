Complexity Analysis

Mathematical Reduction:
Instead of finding a subarray with avg >= `mid` directly, we shift all elements by `-mid`. If the shifted array has a subarray of length >= `k` with sum >= 0, then its original average is >= `mid`.

Time Complexity: O(N * log(MAX_VAL/ε))
The binary search runs a fixed number of iterations (e.g., 70 for ~10^-15 precision). Inside the loop, computing prefix sums and finding the max subarray takes O(N).

Space Complexity: O(N)
An array `pref` of size `N+1` is allocated in each iteration to store prefix sums.
