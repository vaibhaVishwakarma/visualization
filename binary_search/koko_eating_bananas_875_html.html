Complexity Analysis

Core Logic:
The minimum possible eating speed is 1. The maximum speed is the largest pile size (eating faster won't save any more hours per pile). We binary search within this range `[1, max_pile]` to find the minimum speed that satisfies the condition.

Time Complexity: O(N log M)
Where N is the number of piles and M is the maximum pile size. The binary search takes O(log M) steps. In each step, verifying if Koko can finish takes O(N) time.

Space Complexity: O(1)
We only use primitive variables to keep track of speeds and hours.
