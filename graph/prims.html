Complexity Analysis

Time Complexity: O(E log V)
Using an Adjacency List and a Priority Queue, we extract the minimum edge at most E times. Each insertion/extraction operation in the Priority Queue takes O(log V) time.

Space Complexity: O(V + E)
The graph representation (adjacency list) takes O(V + E) space. The Priority Queue and the boolean `inMST` array take O(E) and O(V) space respectively.
