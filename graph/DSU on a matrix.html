Complexity Analysis

Time Complexity: O(N²)
Where N is the grid dimension. Phase 1 iterates over the grid doing DSU operations (nearly O(1) with path compression and size tracking). Phase 2 iterates over the grid again, checking 4 neighbors for each 0. Overall it scales linearly with the number of cells.

Space Complexity: O(N²)
The `parent` and `size` tracking structures (or the Point arrays in the Java code) require memory proportional to the total number of cells in the matrix.
