Complexity Analysis

Time Complexity: O(M * N)
Every cell is processed at most once. Initializing the queue and matrix takes O(M * N). The BFS while-loop pops each cell exactly once, doing a constant O(1) checks (4 directions).

Space Complexity: O(M * N)
The space complexity is dominated by the Queue. In the worst-case scenario (e.g., all 0s), the queue will hold M * N elements simultaneously.
