Algorithm Analysis

Name: Lee Algorithm
A specific application of BFS used to find the shortest path between a source and destination in a uniform grid or maze.

Time Complexity: O(N²)
Where N is the side length of the matrix. The algorithm visits each cell in the matrix at most once, performing 8 constant-time checks for neighbors.

Space Complexity: O(N²)
The `Queue` and the `dist` (visited) array both require O(N²) space in the worst case (e.g., an empty grid where the queue grows very large).
