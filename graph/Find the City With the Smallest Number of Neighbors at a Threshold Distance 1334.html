Complexity Analysis

Time Complexity: O(N³)
The core logic relies on three nested loops iterating from 0 to N. Therefore, the time complexity strictly scales cubically with the number of cities. Finding the best city at the end takes an additional O(N²).

Space Complexity: O(N²)
We use an N×N distance matrix to store the shortest paths between all pairs of nodes.
