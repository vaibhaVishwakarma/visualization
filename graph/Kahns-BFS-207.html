Complexity Analysis

Time Complexity: O(V + E)
Building the adjacency list takes O(E). Enqueueing initial 0-indegree nodes takes O(V). The BFS while loop processes each node exactly once and iterates over all edges exactly once, adding O(V + E).

Space Complexity: O(V + E)
The adjacency list requires O(V + E) space. The `indegree` array and Queue both require O(V) space.
