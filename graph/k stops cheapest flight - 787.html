JAVA CODE (LEVEL-RESTRICTED BFS)


Complexity Analysis

Time Complexity: O((V + E) × K)
The BFS explores layer by layer up to K+1 times. In the worst case, every edge might be checked during each level. The copying of the cost array takes O(V) per level.

Space Complexity: O(V + E)
The adjacency matrix requires O(V²) (or O(V+E) if an adjacency list is used). The cost and nextCost arrays take O(V) space. The Queue takes up to O(V) space.
